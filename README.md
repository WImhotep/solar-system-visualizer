# 太阳系可视化项目

这是一个使用 React 和 TypeScript 开发的太阳系可视化项目，旨在展示太阳系的天体，包括太阳、行星、小行星带等，并提供交互式控制面板让用户可以选择不同的天体进行查看。该项目还包括双语支持（中文和英文）。

## 目录结构





### 主要功能

- **太阳系展示**：通过不同的组件展示太阳、行星、小行星带、星空背景等天体及其运动。
- **天体信息卡**：点击行星或其他天体后，显示详细的信息卡片。
- **行星选择器**：提供一个交互式的行星选择器，允许用户选择不同的天体进行查看。
- **双语支持**：支持中文和英文切换，用户可以根据自己的偏好选择语言。
